💊 Pharma Vision Dashboard

An interactive Excel Dashboard Project built for analyzing pharmacy and healthcare sales data.
This dashboard provides insights into revenue, prescriptions, patients, suppliers, drug availability, expiry status, insurance-based revenue, doctor performance, and sales trends using Excel visualization tools.

📌 Project Overview

The Pharma Vision Dashboard is designed to help pharmacy managers and healthcare analysts monitor:

- Revenue performance
- Drug inventory and expiry status
- Prescription statistics
- Supplier contributions
- Insurance-wise revenue
- Doctor performance
- Gender-based patient distribution
- Sales trends over time

The dashboard uses Excel features such as:

- Pivot Tables
- Pivot Charts
- Slicers
- Interactive Filters
- KPI Cards
- Conditional Formatting

📊 Dashboard Features
🔹 KPI Cards

The dashboard highlights key metrics including:

- Total Revenue
- Average Revenue
- Total Patients
- Total Drugs Available
- Total Prescriptions

🔹 Interactive Filters (Slicers)

Users can filter the dashboard dynamically by:

- Date
- Supplier Name
- Expiry Status
- Gender
- Insurance Provider
- Doctor Name
- Brand Name

🔹 Visual Insights
✅ Expiry Status Count

Tracks expired and safe medicines.

✅ Top Selling Drugs

Displays the most frequently sold medicines.

✅ Drug per Supplier

Shows supplier contribution to drug inventory.

✅ Top Doctors

Analyzes prescription counts by doctors.

✅ Revenue by Insurance

Compares revenue generated from insurance providers.

✅ Sales Trend

Tracks yearly/monthly revenue performance.

✅ Gender Count

Displays gender-wise patient distribution.
